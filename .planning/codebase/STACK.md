# Technology Stack

## Core
- **Language**: Python 3.10+
- **Framework**: FastAPI (Web API)
- **Server**: Uvicorn (ASGI)

## Libraries
- **HTTP Client**: `httpx` (Async HTTP), `openai` (Python SDK)
- **Data Validation**: `pydantic`
- **Environment**: `python-dotenv`
- **Serialization**: `json`, `base64`

## Frontend
- **Framework**: React 18+ (Vite)
- **Styling**: Tailwind CSS (CDN-based for embedded, potentially local for Vite)
- **Icons**: Lucide React (referenced in requirements/skill list)
